1. What is CSS?

CSS means Cascading Style Sheets, which is used to add style the webpages for better user interface and interactive purpose

2. What is mean by Cascading

Cascading in CSS refers to how styles are applied based on source order, specificity, and importance. It allows multiple style rules to affect an element, with the last defined rule taking precedence, more specific selectors overriding less specific ones, and '!important' styles overriding all. This system enables organized and flexible styling.

3. In how many ways we can insert css into html file

There are three ways to insert CSS to HTML file.

  • Inline
  • Internal
  • External

4. which way have more specificity and why

Inline CSS has the highest specificity, overriding internal and external styles. Internal CSS comes next, while external styles have the lowest specificity. Specificity determines which styles apply when multiple rules target the same element.
Inline > Internal > External

5. What is meant by box model.

The box model in CSS describes how elements are structured and how their dimensions are calculated. It consists of the following components:
Content: The actual content of the box (text, images, etc.).
Padding: The space between the content and the border, adding inner spacing.
Border: The line surrounding the padding (if any) and content.
Margin: The space outside the border, creating separation from other elements.

6. What are the values are there in box model

The box model in CSS has several properties that define its values:
Width and Height: Defines the size of the content area.
Padding: Controls the space between the content and the border.
Border: Specifies the width, style, and color of the border surrounding the padding.
Margin: Sets the space outside the border, separating the element from others.
Box-sizing: A property that defines how the width and height of the element are calculated, either including or excluding padding and border.
Values:
content-box (default, excludes padding and border from width/height)
border-box (includes padding and border in width/height)

7. Define Flex

Flex refers to the Flexbox Layout model in CSS, which is designed to help layout elements in a one-dimensional space (either in a row or a column). It allows for flexible and responsive designs by distributing space and aligning items efficiently.
Key features include: Flexible Item Sizes Alignment Control Order Control

8.Advantages of using flex

The advantages of using Flexbox include:
Responsive Design: Easily create layouts that adapt to different screen sizes.
Alignment Control: Simple alignment of items along both main and cross axes.
Space Distribution: Efficiently distribute space among items, allowing for flexible item sizing.
Order Management: Change the visual order of elements without altering HTML.
Simplified Layouts: Reduce the need for complex CSS floats or positioning.

9. Flex properties

There are key Flexbox properties:
display: flex;: Enables Flexbox on the container.
flex-direction: Defines the direction of flex items (row, column, row-reverse, column-reverse).
flex-wrap: Specifies whether flex items should wrap onto multiple lines.
justify-content: Aligns items along the main axis (start, end, center, space-between, space-around).
align-items: Aligns items along the cross axis (stretch, center, start, end, baseline).
align-content:Aligns flex lines when there's extra space in the cross axis (similar values as align-items).

10. What is meant by text property

The text property in CSS refers to a group of properties that control the styling of text within HTML elements. Key properties include:
color: Sets the color of the text.
font-family: Specifies the font used for the text.
font-size: Defines the size of the text.
font-weight: Controls the thickness of the text (e.g., normal, bold).
text-align: Aligns the text (e.g., left, right, center).
line-height: Sets the space between lines of text.
text-decoration: Adds decorations like underline, overline, or strikethrough.
text-transform: Controls text casing (e.g., uppercase, lowercase).

11. Define Font property

The font property in CSS is a shorthand that allows you to set multiple font-related properties in a single declaration. It can include:
font-style: Specifies the style (normal, italic, oblique).
font-variant: Controls the use of small-caps.
font-weight: Defines the thickness of the text (normal, bold, numeric values).
font-size: Sets the size of the font.
line-height: Determines the height of each line of text.
font-family: Specifies the font type (e.g., Arial, Georgia).

12. Difference between text and font

Text refers to the actual content displayed in an HTML element (e.g., words and sentences), while font refers to the typeface and style used to render that text (e.g., Arial, bold). Text styling involves properties like color and alignment, whereas font properties include font-family, size, and weight. In essence, text is the content, and font determines its visual representation.

13. how many types of colors in CSS

In CSS, there are several types of color values:
Named Colors: Predefined color names (e.g., red, blue).
Hexadecimal Colors: Codes starting with #, representing RGB values (e.g., #FF5733).
RGB Colors: Defined using the rgb() function, with values for red, green, and blue (e.g., rgb(255, 87, 51)).
RGBA Colors: Similar to RGB but includes an alpha value for transparency (e.g., rgba(255, 87, 51, 0.5)).
HSL Colors: Defined using the hsl() function based on hue, saturation, and lightness (e.g., hsl(12, 100%, 60%)).
HSLA Colors: Like HSL but includes an alpha value for transparency (e.g., hsla(12, 100%, 60%, 0.5)).
